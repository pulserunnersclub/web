//---- Listas ----//
ol=orded list para crear lista y deospues sus subs contemidos contemidos
li=list item (dentro de li)
ul= lista desordenada (no importa el orden)


// ---- Formularios ----//
form= crear formulario
input= crear objeto y decir su tipo con type=""
type= descrir el tipo de objeto a crear
----->Radio:boton relleno de selccionar(ej:elejir varias opciones)
----->checkbox:caja que al darle click se pone un check(ej:aceptar terminos y condicones)
----->Password:para tapar contraseña
----->email:hacer que no se salte la escritura correcta del example@gmail.com y lo siga como corresponde
----->submit:Boton de enviar que se puede cambiar su texto con: VALUE
----->file:para que el usuario envie documentos de cualquier tipo
----->date:para seleccionar fechas con dia mes y año

// ---- Atributos de el imput ----//
----->required: poner despues del tipo de input para que obligue a llenar ese campo(No es tipo de input)
----->name: para darle un nombre a un campo(ej: un imput con boton de escribir gmail y el input tiene el Atributo
name entonces en la base de datos lo guarda en esa casilla)
----->placeholder: escribir algo dentro de una casilla (ej:escribe aqui tu gmail)
----->minlength: minimo de caracteres a escribir
-------------------------------------------------------------------------------------------------------------------
value= escribar algo predeterminado dentro de una funcion ej:(Dentro de un espacio para escribir que 
ponga escribe aqui o en un boton de enviar cambiarlo por enviar formulario TODO ESTO
DENTRO DEL INPUT)
